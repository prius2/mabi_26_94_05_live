- 음성 기준으로 내부 개발 프로세스 재편, 테스트 서버 제보 처리 전 정식 서버 반영 일정 재조정, QA 보강 등이 개선 방향으로 제시됐다.
- 테스트 서버는 본 서버 캐릭터 환경에 가깝게 만들고 반복 준비 작업을 줄이는 방향이 언급됐다.
